# DevOps Intern Task 4 - Git & GitHub

## Overview
This project demonstrates basic Git and GitHub operations used in DevOps workflows, including repository creation, version control, branching, merging, and pushing changes to GitHub.

## Tools Used
- Git
- GitHub
- Visual Studio Code
- Windows Command Prompt

## Tasks Performed
- Initialized a local Git repository
- Connected the local repository to GitHub
- Created and managed branches
- Committed project changes
- Merged branches
- Pushed the repository to GitHub
